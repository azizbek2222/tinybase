Getting Started

Initialize TinyBase SDK in your project using your unique API Key.

new TinyBase({ apiKey })

Creates a new instance of TinyBase.

import { TinyBase } from "https://tinybase.vercel.app/TinyBase-sdk.js";

const tBase = new TinyBase({
    apiKey: "YOUR_TNB_KEY_HERE"
});

Authentication

Manage users with secure Firebase-powered auth methods.

signUp(email, password)

Register a new user account.

await tBase.signUp('user@example.com', 'securePass123');
signIn(email, password)

Log in an existing user.

await tBase.signIn('user@example.com', 'password');
onAuthChange(callback)

Monitor current user session state.

tBase.onAuthChange((user) => {
    if (user) console.log("User UID:", user.uid);
});

Realtime Database

Manage data with automatic synchronization across all clients.

dbSave(path, data)

Push new data to a specific path with a generated ID.

await tBase.dbSave('musics', { name: 'Shape of You', author: 'Ed Sheeran' });
dbUpdate(path, data)

Update specific fields at the given path.

await tBase.dbUpdate('musics/ID_123', { name: 'Updated Name' });
dbWatch(path, callback)

Listen for real-time changes. Executes callback whenever data changes.

tBase.dbWatch('musics', (data) => {
    console.log("Live Data:", data);
});
dbDelete(path)

Remove data permanently from the database.

await tBase.dbDelete('musics/ID_123');

Financial System

Built-in methods for handling user balances and points.

watchBalance(userId, callback)

Track a user's balance in real-time.

tBase.watchBalance('USER_ID', (balance) => {
    document.getElementById('display').innerText = balance + " $";
});
setBalance(userId, amount)

Update user's balance to a specific numeric value.

await tBase.setBalance('USER_ID', 5000);
