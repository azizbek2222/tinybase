Foydalanuvchi boshqaruvi

signUp / signIn

Loyihangizga kirish va ro'yxatdan o'tish tizimini qo'shish.

// Ro'yxatdan o'tish
await tBase.signUp('email@example.com', 'password');

// Tizimga kirish
await tBase.signIn('email@example.com', 'password');

Universal Ma'lumotlar Bazasi

Ushbu metodlar ixtiyoriy turdagi (mahsulotlar, xabarlar, postlar, musiqalar) ma'lumotlar bilan ishlashga mo'ljallangan.

dbSave(path, data)

Vazifasi: Yangi ma'lumot qo'shish. path - bu papka nomi (kategoriya), data - obyekt.

// Mahsulot qo'shish misoli:
await tBase.dbSave('products', { name: 'iPhone 15', price: 999 });

// Xabar yuborish misoli:
await tBase.dbSave('chats', { user: 'Ali', text: 'Salom!' });

dbUpdate(path, data)

Vazifasi: Mavjud ma'lumotni uning ID-si orqali tahrirlash.

// Ma'lumotni yangilash:
await tBase.dbUpdate('products/ID_KODI', { price: 899 });

dbDelete(path)

Vazifasi: Yo'nalish bo'yicha ma'lumotni o'chirish.

// O'chirish:
await tBase.dbDelete('products/ID_KODI');

dbWatch(path, callback)

Vazifasi: Bazani real-vaqtda kuzatish. Ma'lumot o'zgarsa, ilovangiz avtomatik yangilanadi.

tBase.dbWatch('products', (data) => {
    console.log("Bazadagi yangi ro'yxat:", data);
});

Balans va Hisob-kitob

watchBalance / setBalance

Foydalanuvchining hamyonini yoki ochkolari (points) bilan ishlash.

// Balansni ekranda ko'rsatish
tBase.watchBalance('USER_ID', (summa) => {
    document.getElementById('wallet').innerText = summa;
});

// Balansni o'zgartirish (masalan, yutuq berish)
await tBase.setBalance('USER_ID', 15000);
